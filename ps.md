## 色彩描述

### 三原色
+ 红 黄 蓝

### 三间色
+ 紫 橙 绿
+ 红色 + 蓝色 = 紫色
+ 红色 + 黄色 = 橙色
+ 蓝色 + 黄色 = 绿色

### HSB
+ H：色相（hue）
+ S: 饱和度（saturation）
+ B：亮度（brightness）
